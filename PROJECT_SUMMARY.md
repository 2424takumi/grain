# Grain App v1.0 MVP - Project Summary

## Quick Reference

**Project:** Grain - Privacy-First Photo Diary  
**Timeline:** 8 weeks (56 days)  
**Budget:** $99 (Apple Developer account)  
**Team:** Solo developer  
**Tech Stack:** SwiftUI + SwiftData (iOS 17+)

---

## What We're Building

A simple, privacy-first photo diary app where users take one photo per day with optional text notes. All data stays local on device.

**NOT building in v1.0:**
- App blocking
- Gamification (deferred to v1.1)
- Widgets (deferred to v1.1)
- Notifications (deferred to v1.2)

---

## P0 Features (Must Have)

1. Photo capture (camera integration)
2. Entry creation (photo + text note)
3. One-entry-per-day enforcement
4. Timeline view (all entries in chronological order)
5. Entry detail view (full photo, full note)
6. Edit entry (text note only)
7. Delete entry (with confirmation)
8. Settings screen (basic preferences)
9. Onboarding flow (3 screens)
10. App icon & branding
11. Dark mode support

**Total Estimated:** 132 hours of dev work
